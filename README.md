# pySU
A Python implementation of SU

# Quick Start
### Configure
The configuration file is
* path/to/virtdev/conf/pysu.py

You can add your devices as follows:
* devices={'DEVICE1':'PIN1', 'DEVICE2':'PIN2', ...} 

Example:
>> Two devices:
>> 1) PIR -> pin 'X12'
>> 2) Relay -> pin 'X17'
* devices={'PIR':'X12', 'Relay':'X17'}
